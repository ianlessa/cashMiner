Pequeno projeto cujo objetivo é estudar a linguagem C# e forma de detectar padrões em ativos da bolsa de valores.

O projeto principal é o da pasta Monitor. Este aplicativo foi feito para concentrar plugins de Indicadores e Fontes de Dados (histórios e de tempo real) e, com estes plugins, fazer os backtestings e ajudar a traçar estratégias automatizadas para Daytrade na Bovespa.

No futuro, conterá funcionalidades para avaliar se vale a pena investir em determinado ativo, em determinada quantidade e por determinada corretora.
